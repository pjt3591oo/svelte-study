# svelte speed study

* 프로젝트 생성

```bash
$ npm init svelte@next my-app
$ cd my-app
$ npm i
$ npm run dev
$ npm run build
```

### 특징 & 장점

* 양방향 바인딩 제공

* SSR 제공 

* 디렉터리로 url 구성

  - slug 형태로 path param 전달가능

  - __layout.svelte, __error.svelte와 같은 특수 파일 제공


* props/store 제공

* slot 제공(similar children on react)

* 빠른 빌드타임
